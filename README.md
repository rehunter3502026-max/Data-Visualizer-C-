# Data-Visualizer-C++

A simple console-based data analysis and visualization tool built in C++.
The program allows users to input numerical data, compute basic statistics, and generate a text-based bar chart visualization.

This project demonstrates core Object-Oriented Programming (OOP) concepts and basic data analytics logic in C++.

🚀 Features

Add numerical data dynamically

Display full dataset

Compute basic statistics:

Average

Maximum

Minimum

Generate ASCII-based bar chart visualization

Interactive menu-driven interface

🧠 Concepts Used

Classes & Objects

Encapsulation

Vectors (dynamic arrays)

Loops and conditional logic

Console-based UI

📂 Project Structure
🔹 DataVisualizer Class

Stores dataset using vector<int> and provides:

addData() → Adds user input to dataset

showData() → Displays all stored values

statistics() → Calculates average, max, and min

barChart() → Displays ASCII bar chart

💡 Learning Outcomes

Building CLI-based applications

Implementing statistical calculations manually

Basic data visualization without external libraries

Structuring modular C++ programs
